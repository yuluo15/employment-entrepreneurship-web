## 🔧 数据库表结构

### sys_message 表

```sql
CREATE TABLE sys_message (
  id varchar(64) PRIMARY KEY,
  receiver_id varchar(64) NOT NULL,           -- 接收人ID（学生ID）
  title varchar(200) NOT NULL,                -- 标题
  content text,                               -- 内容
  type integer NOT NULL,                      -- 消息类型（1:系统通知, 2:面试通知, 3:offer通知）
  is_read integer DEFAULT 0,                  -- 阅读状态（0:未读, 1:已读）
  ref_id varchar(64),                         -- 关联业务ID（投递ID）
  create_time timestamp DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_message_receiver ON sys_message(receiver_id);
CREATE INDEX idx_message_type ON sys_message(type);
CREATE INDEX idx_message_read ON sys_message(is_read);
CREATE INDEX idx_message_time ON sys_message(create_time);
```

## 📝 企业端集成说明

### 企业端需要在以下场景创建消息

#### 1. 安排面试时

```java
// 在企业端安排面试的接口中添加
public void scheduleInterview(InterviewDTO dto) {
    // 1. 保存面试信息
    Interview interview = saveInterview(dto);
    
    // 2. 获取投递记录和学生ID
    JobApplication application = applicationMapper.selectById(dto.getApplicationId());
    
    // 3. 获取职位和公司信息
    Position position = positionMapper.selectById(application.getPositionId());
    Company company = companyMapper.selectById(position.getCompanyId());
    
    // 4. 构建消息内容
    String content = String.format(
        "您投递的职位已安排面试\n\n" +
        "职位名称：%s\n" +
        "公司名称：%s\n" +
        "面试时间：%s\n" +
        "面试地点：%s\n" +
        "联系方式：%s\n\n" +
        "备注：%s",
        position.getPositionName(),
        company.getName(),
        dto.getInterviewTime(),
        dto.getInterviewAddress(),
        dto.getContactPhone(),
        dto.getRemark() != null ? dto.getRemark() : "无"
    );
    
    // 5. 创建消息
    MessageEntity message = new MessageEntity();
    message.setId(UUID.randomUUID().toString());
    message.setReceiverId(application.getStudentId());
    message.setTitle("您投递的职位已安排面试");
    message.setContent(content);
    message.setType(2);  // 面试通知
    message.setIsRead(0);
    message.setRefId(dto.getApplicationId());
    message.setCreateTime(new Timestamp(System.currentTimeMillis()));
    
    messageMapper.insert(message);
}
```
