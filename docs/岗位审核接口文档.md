# 岗位审核接口文档

## 一、获取岗位列表

**接口路径**: `GET /api/jobAudit/list`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | Integer | 是 | 页码 |
| pageSize | Integer | 是 | 每页数量 |
| jobName | String | 否 | 职位名称(模糊查询) |
| companyName | String | 否 | 企业名称(模糊查询) |
| audit | Integer | 否 | 审核状态: 0=待审核, 1=已通过, 2=已驳回 |

**返回数据**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 150,
    "data": [
      {
        "jobId": "1",
        "jobName": "Java开发工程师",
        "salaryRange": "SALARY_8K_15K",
        "salaryRangeLabel": "8k-15k",
        "companyId": "1",
        "companyName": "字节跳动",
        "companyLogo": "http://...",
        "companyIndustry": "互联网/IT",
        "city": "成都",
        "location": "高新区天府软件园",
        "education": "BACHELOR",
        "educationLabel": "本科",
        "experience": "1-3年",
        "tags": "五险一金,弹性工作,年终奖",
        "description": "岗位职责...",
        "requirement": "任职要求...",
        "audit": 0,
        "viewCount": 128,
        "hrId": "hr_001",
        "hrName": "张HR",
        "contactPhone": "13800138000",
        "reason": null,
        "createTime": "2026-01-20 14:30:00"
      }
    ]
  }
}
```

**SQL实现**:
```sql
SELECT 
  j.job_id as jobId,
  j.job_name as jobName,
  j.salary_range as salaryRange,
  d1.dict_label as salaryRangeLabel,
  j.company_id as companyId,
  c.name as companyName,
  c.logo as companyLogo,
  c.industry as companyIndustry,
  j.city,
  j.location,
  j.education,
  d2.dict_label as educationLabel,
  j.experience,
  j.tags,
  j.description,
  j.requirement,
  j.audit,
  j.view_count as viewCount,
  j.hr_id as hrId,
  h.name as hrName,
  j.contact_phone as contactPhone,
  j.reason,
  j.create_time as createTime
FROM biz_job j
LEFT JOIN sys_company c ON j.company_id = c.id
LEFT JOIN biz_hr h ON j.hr_id = h.id
LEFT JOIN sys_dict_data d1 ON j.salary_range = d1.dict_value AND d1.dict_type = 'sys_salary'
LEFT JOIN sys_dict_data d2 ON j.education = d2.dict_value AND d2.dict_type = 'sys_education'
WHERE 1=1
  AND (? IS NULL OR j.job_name LIKE CONCAT('%', ?, '%'))
  AND (? IS NULL OR c.name LIKE CONCAT('%', ?, '%'))
  AND (? IS NULL OR j.audit = ?)
ORDER BY j.create_time DESC
LIMIT ?, ?
```

## 二、审核岗位

**接口路径**: `POST /api/jobAudit/audit`

**请求参数**:
```json
{
  "jobId": "1",
  "audit": 1,  // 1=通过, 2=驳回
  "reason": "驳回原因(audit=2时必填)"
}
```

**返回数据**:
```json
{
  "code": 200,
  "message": "success",
  "data": null
}
```

**SQL实现**:
```sql
UPDATE biz_job 
SET audit = ?, 
    reason = ?
WHERE job_id = ?
```

## 三、下架岗位

**接口路径**: `POST /api/jobAudit/offline/{jobId}`

**返回数据**:
```json
{
  "code": 200,
  "message": "success",
  "data": null
}
```

**SQL实现**:
```sql
-- 方案1: 修改审核状态为驳回
UPDATE biz_job 
SET audit = 2,
    reason = '管理员下架'
WHERE job_id = ?

-- 方案2: 如果有单独的status字段表示上下架
UPDATE biz_job 
SET status = 0  -- 0=下架, 1=上架
WHERE job_id = ?
```

## 注意事项

### 1. 字段映射关系

| 数据库字段 | 前端字段 | 说明 |
|-----------|---------|------|
| job_id | jobId | 岗位ID |
| job_name | jobName | 职位名称 |
| salary_range | salaryRange | 薪资范围(字典值) |
| - | salaryRangeLabel | 薪资范围(字典标签) |
| education | education | 学历要求(字典值) |
| - | educationLabel | 学历要求(字典标签) |
| location | location | 详细地址 |
| audit | audit | 审核状态 |
| reason | reason | 驳回原因 |
| hr_id | hrId | HR的ID |
| contact_phone | contactPhone | 联系电话 |

### 2. tags字段处理

`tags` 字段在数据库中是 `varchar` 类型，存储逗号分隔的字符串，如：`"五险一金,弹性工作,年终奖"`

**后端处理**:
```java
// 直接返回字符串即可，前端会自己split
vo.setTags(entity.getTags());
```

**前端处理**:
```typescript
// 前端会这样处理
const tagArray = row.tags.split(',').filter(t => t)
```

### 3. 字典关联

需要关联两个字典表：
- `sys_salary` - 薪资范围字典
- `sys_education` - 学历要求字典

确保字典数据存在：
```sql
-- 薪资范围字典
INSERT INTO sys_dict_data (dict_type, dict_value, dict_label, dict_sort) VALUES
('sys_salary', 'SALARY_3K_5K', '3k-5k', 1),
('sys_salary', 'SALARY_5K_8K', '5k-8k', 2),
('sys_salary', 'SALARY_8K_15K', '8k-15k', 3),
('sys_salary', 'SALARY_15K_25K', '15k-25k', 4),
('sys_salary', 'SALARY_25K_PLUS', '25k以上', 5);

-- 学历要求字典
INSERT INTO sys_dict_data (dict_type, dict_value, dict_label, dict_sort) VALUES
('sys_education', 'HIGH_SCHOOL', '高中', 1),
('sys_education', 'COLLEGE', '大专', 2),
('sys_education', 'BACHELOR', '本科', 3),
('sys_education', 'MASTER', '硕士', 4),
('sys_education', 'DOCTOR', '博士', 5);
```

### 4. HR信息关联

如果 `biz_hr` 表存在，需要关联查询HR姓名：
```sql
LEFT JOIN biz_hr h ON j.hr_id = h.id
```

如果没有 `biz_hr` 表，可以直接从 `sys_user` 表关联：
```sql
LEFT JOIN sys_user u ON j.hr_id = u.id
```

## Java实现示例

### Controller层
```java
@RestController
@RequestMapping("/api/jobAudit")
public class JobAuditController {
    
    @Autowired
    private JobAuditService jobAuditService;
    
    @GetMapping("/list")
    public Result<PageResult<JobVO>> getList(JobQuery query) {
        PageResult<JobVO> result = jobAuditService.getList(query);
        return Result.success(result);
    }
    
    @PostMapping("/audit")
    public Result<Void> audit(@RequestBody AuditJobReq req) {
        jobAuditService.audit(req);
        return Result.success();
    }
    
    @PostMapping("/offline/{jobId}")
    public Result<Void> offline(@PathVariable String jobId) {
        jobAuditService.offline(jobId);
        return Result.success();
    }
}
```

### Service层
```java
@Service
public class JobAuditServiceImpl implements JobAuditService {
    
    @Autowired
    private BizJobMapper jobMapper;
    
    @Override
    public PageResult<JobVO> getList(JobQuery query) {
        // 分页查询
        PageHelper.startPage(query.getPageNum(), query.getPageSize());
        List<JobVO> list = jobMapper.selectJobList(query);
        
        // 处理tags字段 - 直接返回字符串即可
        // 前端会自己split
        
        return PageResult.of(list);
    }
    
    @Override
    public void audit(AuditJobReq req) {
        BizJob job = new BizJob();
        job.setJobId(req.getJobId());
        job.setAudit(req.getAudit());
        job.setReason(req.getReason());
        jobMapper.updateByPrimaryKeySelective(job);
    }
    
    @Override
    public void offline(String jobId) {
        BizJob job = new BizJob();
        job.setJobId(jobId);
        job.setAudit(2);  // 设置为驳回状态
        job.setReason("管理员下架");
        jobMapper.updateByPrimaryKeySelective(job);
    }
}
```
