# 企业端工作台接口文档

## 1. 获取工作台数据

**接口地址**: `GET /api/company/dashboard`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**: 无

**响应示例**:

**字段说明**:
- `companyName`: 企业名称
- `companyLogo`: 企业LOGO地址
- `hrName`: 当前登录HR姓名
- `activeJobCount`: 在招职位数（status=1 且 audit=1）
- `pendingResumeCount`: 待处理简历数（status='DELIVERED'）
- `todayInterviewCount`: 今日面试数（面试日期=今天 且 status=0）
- `monthResumeCount`: 本月简历数（本月投递的简历总数）
- `pendingResumes`: 待处理简历列表（最新5条，按投递时间倒序）
- `todayInterviews`: 今日面试列表（按面试时间正序）
- `resumeTrend`: 近7天简历投递趋势
  - `dates`: 日期数组（格式：MM-DD）
  - `counts`: 对应日期的简历数量
- `jobRank`: 职位投递排行TOP5（按投递数量倒序）

**面试类型说明**:
- `1`: 线下面试
- `2`: 视频面试
- `3`: 电话面试

## 后端实现参考

### SQL查询示例

```sql
-- 1. 获取企业和HR基本信息
SELECT 
  c.name AS company_name,
  c.logo AS company_logo,
  h.name AS hr_name
FROM sys_company c
JOIN biz_hr h ON h.company_id = c.id
WHERE h.user_id = #{userId};

-- 2. 在招职位数
SELECT COUNT(*) 
FROM biz_job 
WHERE company_id = #{companyId} 
  AND status = 1 
  AND audit = 1;

-- 3. 待处理简历数
SELECT COUNT(*) 
FROM biz_job_delivery d
JOIN biz_job j ON d.job_id = j.id
WHERE j.company_id = #{companyId} 
  AND d.status = 'DELIVERED';

-- 4. 今日面试数
SELECT COUNT(*) 
FROM biz_interview i
JOIN biz_job j ON i.job_id = j.id
WHERE j.company_id = #{companyId}
  AND DATE(i.interview_time) = CURRENT_DATE
  AND i.status = 0;

-- 5. 本月简历数
SELECT COUNT(*) 
FROM biz_job_delivery d
JOIN biz_job j ON d.job_id = j.id
WHERE j.company_id = #{companyId}
  AND DATE_TRUNC('month', d.create_time) = DATE_TRUNC('month', CURRENT_DATE);

-- 6. 待处理简历列表（最新5条）
SELECT 
  d.id,
  u.real_name AS student_name,
  u.avatar AS student_avatar,
  j.job_name,
  TO_CHAR(d.create_time, 'YYYY-MM-DD HH24:MI:SS') AS create_time
FROM biz_job_delivery d
JOIN biz_job j ON d.job_id = j.id
JOIN sys_user u ON d.student_id = u.user_id
WHERE j.company_id = #{companyId}
  AND d.status = 'DELIVERED'
ORDER BY d.create_time DESC
LIMIT 5;

-- 7. 今日面试列表
SELECT 
  i.id,
  u.real_name AS student_name,
  j.job_name,
  TO_CHAR(i.interview_time, 'YYYY-MM-DD HH24:MI') AS interview_time,
  i.type
FROM biz_interview i
JOIN biz_job j ON i.job_id = j.id
JOIN sys_user u ON i.student_id = u.user_id
WHERE j.company_id = #{companyId}
  AND DATE(i.interview_time) = CURRENT_DATE
  AND i.status = 0
ORDER BY i.interview_time ASC;

-- 8. 近7天简历投递趋势
SELECT 
  TO_CHAR(date_series, 'MM-DD') AS date,
  COALESCE(COUNT(d.id), 0) AS count
FROM (
  SELECT generate_series(
    CURRENT_DATE - INTERVAL '6 days',
    CURRENT_DATE,
    '1 day'::interval
  )::date AS date_series
) dates
LEFT JOIN biz_job_delivery d ON DATE(d.create_time) = dates.date_series
LEFT JOIN biz_job j ON d.job_id = j.id AND j.company_id = #{companyId}
GROUP BY dates.date_series
ORDER BY dates.date_series ASC;

-- 9. 职位投递排行TOP5
SELECT 
  j.job_name,
  COUNT(d.id) AS count
FROM biz_job j
LEFT JOIN biz_job_delivery d ON j.id = d.job_id
WHERE j.company_id = #{companyId}
  AND j.status = 1
  AND j.audit = 1
GROUP BY j.id, j.job_name
ORDER BY count DESC
LIMIT 5;
```

### Java Controller示例

```java
@RestController
@RequestMapping("/api/company")
public class CompanyDashboardController {

    @Autowired
    private CompanyDashboardService dashboardService;

    @GetMapping("/dashboard")
    public Result<DashboardVO> getDashboard() {
        // 从token获取当前用户ID
        String userId = SecurityUtils.getUserId();
        
        // 调用service获取工作台数据
        DashboardVO data = dashboardService.getDashboardData(userId);
        
        return Result.success(data);
    }
}
```

### VO类定义

```java
@Data
public class DashboardVO {
    private String companyName;
    private String companyLogo;
    private String hrName;
    private Integer activeJobCount;
    private Integer pendingResumeCount;
    private Integer todayInterviewCount;
    private Integer monthResumeCount;
    private List<PendingResumeVO> pendingResumes;
    private List<TodayInterviewVO> todayInterviews;
    private ResumeTrendVO resumeTrend;
    private List<JobRankVO> jobRank;
}

@Data
public class PendingResumeVO {
    private String id;
    private String studentName;
    private String studentAvatar;
    private String jobName;
    private String createTime;
}

@Data
public class TodayInterviewVO {
    private String id;
    private String studentName;
    private String jobName;
    private String interviewTime;
    private Integer type;
}

@Data
public class ResumeTrendVO {
    private List<String> dates;
    private List<Integer> counts;
}

@Data
public class JobRankVO {
    private String jobName;
    private Integer count;
}
```

## 注意事项

1. 所有查询都需要基于当前登录HR所属的企业ID进行过滤
2. 待处理简历只统计status='DELIVERED'的记录
3. 今日面试只统计面试日期为今天且status=0（待进行）的记录
4. 简历趋势需要补全7天数据，没有数据的日期count为0
5. 职位排行只统计在招且已通过审核的职位
6. 时间格式统一使用 `YYYY-MM-DD HH24:MI:SS` 或 `YYYY-MM-DD HH24:MI`
