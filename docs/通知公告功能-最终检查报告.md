## 🎯 核心设计要点

### 1. 数据库字段使用

```sql
-- 管理员发布
publisher_type = 'admin'
publisher_id = NULL
target_audience = 'all' 或 'student'
is_top = 0 或 1

-- 学校发布
publisher_type = 'school'
publisher_id = 学校ID
target_audience = 'student'
is_top = 0
```

### 2. 权限控制SQL

```sql
-- 学校端查询（能看到管理员的+本校的）
WHERE (
  (publisher_type = 'admin' AND status = 1)
  OR (publisher_type = 'school' AND publisher_id = #{schoolId})
)

-- 学校端更新/删除（只能操作本校的）
WHERE notice_id = #{noticeId}
  AND publisher_type = 'school'
  AND publisher_id = #{schoolId}

-- 学生端查询（能看到管理员面向学生的+本校的）
WHERE status = 1
  AND (
    (publisher_type = 'admin' AND target_audience IN ('all', 'student'))
    OR (publisher_type = 'school' AND publisher_id = #{studentSchoolId})
  )
```

### 3. 前端权限判断
