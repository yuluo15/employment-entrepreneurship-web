### 2. 学校端接口实现

#### GET /api/school/notice/list
```sql
-- 关键查询条件
WHERE (
  (n.publisher_type = 'admin' AND n.status = 1)
  OR (n.publisher_type = 'school' AND n.publisher_id = #{schoolId})
)
```

#### POST /api/school/notice/add
```sql
-- 关键字段设置
publisher_type = 'school'
publisher_id = #{schoolId}
target_audience = 'student'
is_top = 0
```

#### PUT /api/school/notice/update
```sql
-- 关键权限校验
WHERE notice_id = #{noticeId}
  AND publisher_type = 'school'
  AND publisher_id = #{schoolId}
```

#### DELETE /api/school/notice/delete/{noticeId}
```sql
-- 关键权限校验
WHERE notice_id = #{noticeId}
  AND publisher_type = 'school'
  AND publisher_id = #{schoolId}
```

#### PUT /api/school/notice/publish/{noticeId}
```sql
-- 关键权限校验
WHERE notice_id = #{noticeId}
  AND publisher_type = 'school'
  AND publisher_id = #{schoolId}
  AND status = 0
```

#### PUT /api/school/notice/unpublish/{noticeId}
```sql
-- 关键权限校验
WHERE notice_id = #{noticeId}
  AND publisher_type = 'school'
  AND publisher_id = #{schoolId}
  AND status = 1
```

## 🔍 关键点检查

### 数据库字段使用

| 字段 | 管理员 | 学校 | 说明 |
|------|--------|------|------|
| publisher_type | ✅ 'admin' | ✅ 'school' | 发布者类型 |
| publisher_id | ✅ NULL | ✅ 学校ID | 发布者ID |
| target_audience | ✅ 'all'/'student' | ✅ 'student' | 目标受众 |
| create_by | ✅ 用户ID | ✅ 用户ID | 创建人 |
| is_top | ✅ 可设置 | ✅ 固定0 | 是否置顶 |

### 权限控制

| 操作 | 管理员 | 学校 | 学生 |
|------|--------|------|------|
| 查看全部 | ✅ | ❌ | ❌ |
| 查看管理员公告 | ✅ | ✅ | ✅ |
| 查看本校公告 | ✅ | ✅ | ✅ |
| 新增公告 | ✅ | ✅ | ❌ |
| 编辑管理员公告 | ✅ | ❌ | ❌ |
| 编辑本校公告 | ✅ | ✅ | ❌ |
| 删除管理员公告 | ✅ | ❌ | ❌ |
| 删除本校公告 | ✅ | ✅ | ❌ |
| 置顶公告 | ✅ | ❌ | ❌ |

### 可见性规则

| 公告类型 | 学生可见条件 |
|---------|-------------|
| 管理员发布 | target_audience='all' 或 'student' |
| 本校发布 | publisher_id=学生学校ID |

## ⚠️ 注意事项

### 1. 学校ID获取

学校端接口需要从登录用户获取学校ID：
```java
// 从登录用户的 owner_id 获取学校ID
String schoolId = SecurityUtils.getCurrentUser().getOwnerId();
```

### 2. 学生学校ID获取

学生端接口需要查询学生所属学校：
```java
// 从 biz_student 表获取学生学校ID
String studentSchoolId = studentMapper.getSchoolIdByUserId(userId);
```

### 3. 权限校验

所有学校端的更新/删除操作必须加权限校验：
```sql
WHERE notice_id = #{noticeId}
  AND publisher_type = 'school'
  AND publisher_id = #{schoolId}
```

如果影响行数为0，返回错误"无权限操作"。

### 4. 附件字段

数据库表中的 `attachments` 字段不再使用，但保留不删除。
